# KoboldKeyboard

All files for my changes to the keyboard I bought as a kit some time ago.

It it a RGB Sofle rev2 from Keyhive.

Features activated:

- OLEDs
- Backlight and per-ley LEDs
- some animations
- German Umlaute by pressing two keys simultaniously 
    (works only if OS keyboard setting is English International)
- right side is main and must be connected to the computer
